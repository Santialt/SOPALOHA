# SOPALOHA Web

Frontend interno de soporte tecnico POS/Aloha.

## Requisitos

- Node.js 18+
- Backend API corriendo en `http://localhost:3001`

## Correr localmente

```bash
cd apps/web
npm install
npm run dev
```

La app queda disponible en `http://localhost:5173`.

## Config opcional

Si necesitŠs otro backend, podťs definir:

```bash
VITE_API_BASE_URL=http://localhost:3001
```
